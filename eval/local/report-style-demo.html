Methods

Report structure intended for generated eval output

This demo separates the report into manuscript-like sections: abstract, headline effects, figures, tabular appendix, and source notes. The shape maps cleanly onto the current eval JSON fields without requiring a new data model.

  • Scores are shown as means on a 0.00-1.00 scale, with deltas centered on the baseline.
  • Costs and tokens remain visible, but they are framed as a trade-off rather than mixed into quality scores.
  • Failure attribution is compact evidence text and can later expand into linked raw logs.
  • The static HTML avoids external assets so CI artifacts remain self-contained.
